## Qilin 麒麟

基于 react, redux, webpack 的一款轻量级前端应用框架

---

## 特性

- **统一规范**，统一的代码, commit, git flow 规范(开发中...)
- **化繁为简**，简化数据流状态管理方案
- **个性化风格**，仅少量必要配置

## 快速上手

```bash
# 创建目录
$ mkdir myapp && cd myapp

# 用工具创建项目
$ yarn create @qilinjs/qilin-app
# 或 npx @qilinjs/create-qilin-app

# 安装依赖
$ yarn

# 启动开发
$ yarn start
```
